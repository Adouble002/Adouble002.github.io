Currency Calculator

Video of Currency Calculator

3a Response

The overall purpose of the program is to convert the United States dollar into the South Korean won. The functionality of the program in the demonstrated video is to take the number that is inputted in the entry box and to times it by the conversion number of US dollars to South Korean won and finally to display it in the output box. The input and output of the program are the entry box, the top box with the clear button next to it, and the output is the box below it. What happens is that the user puts in a number into the input box and that number is converted using the conversion number to how many South Korean wons would equal to that many dollars.

3b Response

The list name is “names”. The data that is contained in the list is the output that is created after the conversion. It is then put in the “file.txt”. If they wish to open the file they just press file then open and it opens the file. The list is a necessity in the code because if a user needed to calculate the price of multiple items to then add them up, thy would need to save the answers so they could calculate all the prices.
